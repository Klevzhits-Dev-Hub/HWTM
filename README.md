# 💰 Who Wants to Be a Millionaire

A mobile quiz game inspired by the iconic TV show.  
Answer 15 questions, use lifelines, and make it to the million!  
Built from scratch in SwiftUI — no third-party libraries.

---

## 🚀 Features

- 📺 15 questions with increasing difficulty  
- 🧠 Lifelines:  
  - 50/50  
  - Ask the Audience  
  - Phone a Friend  
- 🔊 Sound effects  
- ⏱ Answer timer  
- 💾 High score saving  
- 📱 Adaptive interface for iPhone
